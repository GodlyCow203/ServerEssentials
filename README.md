## What is **EssentialsC**?

**EssentialsC** is a lightweight yet powerful Minecraft plugin inspired by **EssentialsX**, but built with significantly more features, deeper customization, and a strong focus on performance.

It is designed for server owners who want full control over their server systems without sacrificing speed, stability, or simplicity.

---

## Why choose **EssentialsC** over ~~EssentialsX~~?

Unlike traditional all-in-one plugins, **EssentialsC** is built to be:

- Faster and lighter, with no unnecessary bloat  
- Massively configurable, giving full control over every system  
- Modern and flexible, designed for today’s servers  
- Feature-rich while remaining highly optimized  

Whether you run a small SMP or a large network, EssentialsC scales effortlessly to meet your needs.

---

## Key Features

- 20+ dedicated SQLite databases for efficient and organized data storage  
- Full language support with 10+ languages available and growing  
- 100% MiniMessage support for complete text customization  
- Massive configuration system allowing fine-grained control over all features  
- Weekly updates with new features, improvements, and optimizations  
- Actively optimized systems built for long-term stability and performance  

---

## What about bugs?

EssentialsC is actively maintained and thoroughly tested to ensure stability and reliability.

While no plugin can guarantee a completely bug-free experience, issues are addressed quickly, and regular updates help keep your server running smoothly.

---

## Dependencies ( Optional )

EssentialsC requires the following dependencies to function properly:

- [Vault](https://www.spigotmc.org/resources/vault.34315/)  
- [LuckPerms](https://luckperms.net/download)  
- [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)  

Please ensure all dependencies are installed for full functionality.

Download **EssentialsC** and jump right in.  
It is powerful, easy to use, and fully customizable, making it suitable for servers of any size.
